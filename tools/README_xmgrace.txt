
XMGRACE
 Utility to plot and postprocess (fit, interpolate, etc) data.
 Available on Unix/Linux machines (though not always as a default)
 Very convenient to have data files formatted column-wise
 #'s can be use to add comments
 .dat extension allows vim to highlight comments

Usage: 
  xmgrace  [<file1.dat> ... <fileN.dat>]

Impost data:
  Data -> import -> ascii

Fit: 
  Data -> transformation -> non-linear curve fitting

Interpolation:
  Data -> transformation -> interpolation/splines  (use cubic)

Extremely flexible tool, high quality plots, provides extensive control
over graphs

No 3D support (no isosurfaces, no contour lines, etc)
